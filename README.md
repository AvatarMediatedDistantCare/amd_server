# SCOPE Server

## Start
### Development
```sh
$ npm run-script develop
```

### Production (Demonize)
```sh
# start
$ npm run-script production_start

# stop
$ npm run-script production_stop
```

## Requirements
* Ubuntu 14.04 LTS or later
* Node.js v0.10.x or later
  * npm
  * bower
* Nginx v1.3.14 or later
